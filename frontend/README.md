# QuickNote_Frontend

## Project Setup

```sh
npm install
```

### Set enviroment variables in .env
```
VITE_PORT=< port >
VITE_BACKEND_SERVER=< ip:port >
```
### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

### Format

```sh
npm run format
```
